### Página de Login e Registro

Este projeto é uma aplicação full-stack com autenticação.

#### Tecnologias
- Frontend: React, Vite, TypeScript, Tailwind CSS
- Backend: Node.js, Express, TypeScript
- Banco de Dados: SQLite

#### Como rodar o projeto
1. Instale as dependências: `npm install`
2. Inicie o backend: `npm run dev:server` (ou o comando configurado no seu package.json)
3. Inicie o frontend: `npm run dev`# Contorn
